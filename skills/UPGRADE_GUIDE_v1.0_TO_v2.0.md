## 重大变更 / Breaking Changes

### 1. 配置文件结构变更 / Configuration Structure Changes

#### v1.0 配置 / v1.0 Configuration
```json
{
  "name": "my-skill",
  "version": "1.0.0",
  "description": "A sample skill",
  "author": "Developer"
}
```

#### v2.0 配置 / v2.0 Configuration
```json
{
  "name": "my-skill",
  "version": "2.0.0",
  "description": "A sample skill",
  "author": "Developer",
  "metadata": {
    "apiVersion": "2.0",
    "compatibility": {
      "minVersion": "2.0.0",
      "maxVersion": "3.0.0"
    },
    "dependencies": [],
    "tags": ["utility", "automation"]
  }
}
```

## 常见问题 / Troubleshooting

### 问题 1: 配置验证失败

**错误信息:** Missing required field: metadata.apiVersion

**解决方案:**
```json
{
  "metadata": {
    "apiVersion": "2.0",
    "compatibility": {
      "minVersion": "2.0.0"
    }
  }
}
```

### 问题 2: 批量操作内存不足

**解决方案:**
```bash
node --max-old-space-size=4096 ./scripts/batch-tool-integration.js
```
