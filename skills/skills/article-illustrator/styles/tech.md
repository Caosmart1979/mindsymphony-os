# Tech 风格详细参数

## 配色方案

- **主色**：深蓝 (#0A1628)
- **辅色**：紫罗兰 (#6366F1)
- **点缀**：霓虹青 (#00F0FF)、霓虹粉 (#FF006E)
- **中性色**：深灰 (#1E293B)、浅灰 (#CBD5E1)

## 视觉元素

- 电路纹理
- 数据流线条
- 几何图形（六边形、三角形）
- 发光效果
- 网格背景

## 构图风格

- 居中对齐，强调对称
- 多层次叠加
- 纵深感强烈
- 元素间有连接线暗示关联

## 字体建议

- 标题：无衬线粗体（如 Inter Bold）
- 正文：等宽字体（如 JetBrains Mono）

## 适用场景

- 科技话题
- AI/机器学习
- 编程/代码
- 数据分析
- 技术原理
- 系统架构

## 不适用场景

- 情感类内容
- 自然/环保话题
- 历史/怀旧内容

## 示例提示词模板

```
Create a tech-style illustration about [TOPIC]:
- Use deep blue background with neon cyan and magenta accents
- Include circuit board patterns and data flow lines
- Add geometric shapes (hexagons, triangles)
- Create glowing effects for key elements
- Symmetrical composition with strong depth
- Style: futuristic, digital, high-tech
```
