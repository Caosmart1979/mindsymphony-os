# Retro 风格详细参数

## 配色方案

- **主色**：复古橙 (#E76F51)
- **辅色**：怀旧的黄 (#E9C46A)
- **点缀**：复古绿 (#2A9D8F)、怀旧红 (#D62828)
- **中性色**：米黄 (#F4A261)、深褐 (#264653)

## 视觉元素

- 像素风
- 老式插画
- 复古海报
- 怀旧元素
- 复古纹理

## 构图风格

- 复古布局
- 海报风格
- 怀旧氛围
- 经典构图

## 字体建议

- 标题：复古装饰体（如 VT323）
- 正文：像素风格（如 Press Start 2P，小字号）

## 适用场景

- 怀旧话题
- 历史回顾
- 经典重温
- 复古风格
- 时光记忆

## 不适用场景

- 前沿科技
- 未来趋势
- 现代设计

## 示例提示词模板

```
Create a retro-style illustration about [TOPIC]:
- Use vintage orange, yellow, and teal palette
- Include pixel art or old-school illustration elements
- Add nostalgic textures and retro patterns
- Vintage poster-style composition
- Nostalgic atmosphere
- Style: retro, vintage, nostalgic, classic
```
