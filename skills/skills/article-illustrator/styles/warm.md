# Warm 风格详细参数

## 配色方案

- **主色**：暖橙 (#FF6B35)
- **辅色**：金黄 (#F7C59F)
- **点缀**：柔粉 (#FFB5A7)、奶油色 (#FFF8E7)
- **中性色**：暖灰 (#E8D5C4)、米白 (#FAF3E0)

## 视觉元素

- 手绘笔触
- 圆润形状
- 生活场景元素
- 柔和渐变
- 温暖光影

## 构图风格

- 自由流动，不对称
- 元素间有呼吸感
- 亲密视角
- 温馨氛围

## 字体建议

- 标题：圆润无衬线（如 Nunito Bold）
- 正文：人文无衬线（如 Lora）

## 适用场景

- 情感话题
- 个人成长
- 生活感悟
- 人文关怀
- 温暖故事

## 不适用场景

- 严肃技术文档
- 商业分析
- 科学原理

## 示例提示词模板

```
Create a warm-style illustration about [TOPIC]:
- Use warm orange and golden yellow palette
- Include hand-drawn elements with soft brush strokes
- Add rounded shapes and organic forms
- Create gentle gradients and warm lighting
- Intimate, cozy composition
- Style: warm, human, emotional, approachable
```
