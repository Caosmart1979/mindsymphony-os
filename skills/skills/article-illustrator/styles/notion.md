# Notion 风格详细参数

## 配色方案

- **主色**：莫兰迪蓝 (#A4C3D9)
- **辅色**：柔粉 (#E8B4BC)、淡黄 (#F5E6CA)
- **点缀**：薄荷绿 (#C8E6C9)、淡紫 (#D7C4E8)
- **中性色**：灰白 (#F8F9FA)、深灰 (#495057)

## 视觉元素

- 线稿风格
- 图标符号
- 扁平化设计
- 卡片元素
- 简洁插图

## 构图风格

- 网格化布局
- 清晰层次
- 信息图表风
- 易于扫描

## 字体建议

- 标题：清晰无衬线（如 Inter SemiBold）
- 正文：易读无衬线（如 Inter Regular）

## 适用场景

- 知识整理
- 笔记系统
- 方法框架
- 模型介绍
- 思维工具

## 不适用场景

- 情感表达强烈的内容
- 高度抽象的哲学话题

## 示例提示词模板

```
Create a notion-style illustration about [TOPIC]:
- Use muted, pastel color palette
- Include line-drawn icons and symbols
- Flat design with card-like elements
- Grid-based, organized layout
- Clean, informational style
- Style: notion-like, organized, clear, diagrammatic
```
