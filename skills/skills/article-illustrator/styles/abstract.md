# Abstract 风格详细参数

## 配色方案

- **主色**：高饱和渐变（紫红到青蓝）
- **辅色**：霓虹色组合
- **点缀**：金属光泽、全息效果
- **中性色**：深空黑 (#0A0A0A)、星际紫 (#1A0B2E)

## 视觉元素

- 流体形态
- 几何组合
- 超现实元素
- 全息效果
- 维度扭曲

## 构图风格

- 超现实构图
- 多维空间
- 视觉冲击
- 梦幻氛围

## 字体建议

- 标题：未来感无衬线（如 Orbitron Bold）
- 正文：科技感等宽（如 Fira Code）

## 适用场景

- 前沿概念
- 元宇宙
- 未来科技
- 虚拟现实
- 想象力话题

## 不适用场景

- 日常实用内容
- 传统行业话题

## 示例提示词模板

```
Create an abstract-style illustration about [TOPIC]:
- Use high-saturation gradients with neon accents
- Include fluid forms and surreal geometric combinations
- Add holographic effects and dimensional twists
- Dreamlike, avant-garde composition
- Futuristic atmosphere
- Style: abstract, surreal, futuristic, visionary
```
