# 批判思维扩展包 (Thinking Extension)

> 多角度审视、风险识别、深度思辨。让思考更全面、更深刻。

## 技能列表

### 思辨工具（4个）

| 技能 | 定位 | 类型 | 原编号 |
|------|------|------|--------|
| **paradox-reveal** | 矛盾揭示 | analytical | c-02 |
| **dialectic-analysis** | 辩证分析 | analytical | c-04 |
| **extreme-logic** | 极限逻辑 | execution | c-05 |

### 视角透镜（8个）

| 技能 | 视角 | 类型 | 原编号 |
|------|------|------|--------|
| **risk-paranoid** | 焦虑型领导者（风险担忧） | analytical | g-01 |
| **trend-timing** | 趋势投机者（时机评估） | analytical | g-02 |
| **data-driven** | 数据参数派（量化判断） | analytical | g-03 |
| **authority-citing** | 大师搬运工（权威引用） | analytical | g-04 |
| **nostalgia-lens** | 黄金时代守护者（怀旧视角） | analytical | g-05 |
| **niche-critique** | 小众优越者（批判大众） | analytical | g-06 |
| **harsh-truth** | 严厉唤醒者（直言真相） | analytical | g-07 |
| **plain-wisdom** | 街头哲学家（大白话智慧） | analytical | g-08 |

### 道法术器思辨（3个）

| 技能 | 定位 | 类型 | 原编号 |

## 使用场景

| 场景 | 推荐组合 |
|------|----------|
| "审视这个方案的风险" | risk-paranoid + data-driven |
| "用不同角度分析这个问题" | 多个视角透镜组合 |
| "这个概念有什么根本矛盾" | paradox-reveal + dialectic-analysis |
| "用大白话解释这个" | plain-wisdom |
