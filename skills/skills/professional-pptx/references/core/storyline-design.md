# Storyline Design Guide

演讲故事线设计方法论。

---

## 故事线 = 演讲的骨架

```
好的PPT不是信息的堆砌，而是有节奏的叙事

开场钩子 → 建立问题 → 展示方案 → 证明效果 → 行动呼吁
```

---

## 常用叙事结构

### 1. 问题-解决型 (学术/商业通用)

```
▸ 问题 (Why)
  "当前存在什么问题？"
  
▸ 方法 (How)
  "我们如何解决？"
  
▸ 结果 (What)
  "我们发现了什么？"
  
▸ 意义 (So What)
  "这有什么重要性？"
```

**适用场景**: 学术报告、产品发布、解决方案展示

### 2. 时间线型

```
▸ 背景 (过去)
  "我们从哪里来"
  
▸ 过程 (过程)
  "我们做了什么"
  
▸ 现状 (现在)
  "我们在哪里"
  
▸ 展望 (未来)
  "我们要去哪里"
```

**适用场景**: 项目汇报、年度总结、里程碑回顾

### 3. 金字塔型 (商业汇报)

```
▸ 结论 (What)
  先给答案
  
▸ 支撑论点 (Why)
  3个关键理由
  
▸ 详细数据 (How)
  证据和细节
```

**适用场景**: 高管汇报、投资路演、决策会议

### 4. 对比型

```
▸ 现状 (A)
  "目前是什么情况"
  
▸ 问题/痛点
  "存在什么问题"
  
▸ 方案对比 (B vs C)
  "有哪些选择"
  
▸ 推荐
  "我们建议什么"
```

**适用场景**: 方案选择、竞品分析、策略对比

### 5. 教学型

```
▸ 概念 (What)
  "这是什么"
  
▸ 原理 (Why)
  "为什么这样"
  
▸ 示例 (How)
  "具体怎么做"
  
▸ 练习/应用
  "你来试试"
```

**适用场景**: 培训、科普、技术分享

## 开场技巧

### 钩子类型

| 类型 | 示例 | 适用场景 |
|-----|------|---------|
| 提问 | "如果我告诉你..." | 通用 |
| 数据冲击 | "每年有X万人..." | 医学/商业 |
| 故事 | "去年，一位患者..." | 医学/教育 |
| 对比 | "10年前 vs 现在" | 技术/商业 |
| 悬念 | "接下来15分钟将改变..." | TED风格 |

### 开场示例

**学术报告**:
> "胃癌是全球第三大癌症死因。每年新增超过100万例，
> 而我们今天要讨论的，是如何让更多患者获得手术机会。"

**商业汇报**:
> "上个季度，我们的用户流失率达到了15%。
> 今天我要分享的方案，可以把这个数字降到5%以下。"

---

## 结尾技巧

### 结尾模式

1. **总结型** - 重复核心要点
2. **呼吁型** - 告诉听众下一步做什么
3. **展望型** - 描绘未来图景
4. **回扣型** - 回应开场的问题/故事

### 结尾示例

**学术报告**:
> "总结今天的报告：
> 1. 新技术显著降低了术后并发症
> 2. 高龄患者可能获益更多
> 3. 需要多中心研究进一步验证
> 
> 感谢各位的聆听，欢迎提问。"

**商业汇报**:
> "如果我们在下个月启动这个项目，
> 预计Q3就能看到用户留存率的显著提升。
> 我建议立即组建跨部门工作组，下周开始第一阶段。"

## 故事线检查清单

- [ ] 有明确的核心信息
- [ ] 开场能抓住注意力
- [ ] 逻辑线清晰可循
- [ ] 有高潮/转折点
- [ ] 结尾有力量
- [ ] 时间分配合理
- [ ] 每个部分都支撑核心信息
