## 配色使用规则

### 颜色数量
- 主色: 1个
- 辅助色: 1-2个
- 强调色: 1个
- **总计不超过4-5种**

### 对比度要求
- 文字与背景: 4.5:1以上
- 大标题与背景: 3:1以上
- 图表颜色互相: 清晰区分

### 语义一致性
| 含义 | 建议颜色 |
|-----|---------|
| 正面/增长/显著 | 绿色系 |
| 负面/下降/风险 | 红色系 |
| 中性/基准 | 灰色系 |
| 重点/当前 | 蓝色/品牌色 |
| 警示/注意 | 橙色/黄色 |

## ⚠️ 颜色代码注意

**在PptxGenJS中，颜色代码不带#号！**

```javascript
// ✅ 正确
chartColors: ["2b6cb0", "e53e3e"]
fill: { color: "1a365d" }

// ❌ 错误
chartColors: ["#2b6cb0", "#e53e3e"]
fill: { color: "#1a365d" }
```
