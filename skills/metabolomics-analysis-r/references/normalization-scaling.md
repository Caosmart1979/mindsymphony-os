# Normalization, Scaling, Missingness

## Missingness
- MCAR/MAR: kNN or random forest imputation
- MNAR: half-minimum or small constant

## Normalization
- Pre-acquisition: mass/volume/cell count normalization
- Post-acquisition: PQN, median, sum, quantile

## Transform and scale
- Log transform for skewed data
- Auto-scaling or pareto scaling
