# 🚀 AI Director - 快速开始

## 30 秒快速上手

### 1️⃣ 安装依赖
```bash
pip install anthropic
```

### 2️⃣ 设置 API Key
```bash
export ANTHROPIC_API_KEY="your-key-here"
```

### 3️⃣ 运行
```bash
python run.py
```

然后开始对话! 🎉

## ❓ 常见问题

**Q: 需要 API Key 吗?**  
A: 是的,需要从 [Anthropic](https://console.anthropic.com/) 获取

**Q: 支持哪些模型?**  
A: 推荐 Claude Sonnet 4,也支持其他 Claude 模型

**Q: 对话会被保存吗?**  
A: 仅在内存中保存,重启后丢失,可用 export_session() 导出
